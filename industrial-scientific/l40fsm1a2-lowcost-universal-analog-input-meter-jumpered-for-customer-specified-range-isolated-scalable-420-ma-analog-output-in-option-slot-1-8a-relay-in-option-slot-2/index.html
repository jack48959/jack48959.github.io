Low-cost, universal analog input, 4-digit LED, 1/8 DIN panel meter. Includes universal 18-265V AC/DC power Selectable DC and True RMS ranges from 60.0 mV to 400V, 20.00 mA to 5.00A. Selectable thermocouple types K, J, E, N, L, C, R, S, B, T. Selectable RTD types Pt100, Pt500, Pt1000, Ni100, Ni200, Ni1000.

Model L40-FS-M1-A2 by Laurel Electronics is a low-cost universal analog input 4-digit digital panel meter jumpered for a customer specified range, with an isolated 4-20 mA analog output in Slot 1, 8A relay in Slot 2. The L40 provides 65 user-selectable analog input signal types and ranges. Ranges include DC and AC RMS voltage from 60 mV to 400V, DC and AC RMS current from 20 mA to 5A, process signals (4-20 mA, 0-10V), thermocouples (types K, J, E, N, L, C, R, S, B, T, deg C or F), RTD (6 types, deg C or F), thermistors (NTC or PTC), resistance (0-5k or 0-50k ohms), and potentiometer inputs. The same meter handles all of signal types with no need for a plug-in signal conditioner board. Typical accuracy is 0.2% of full scale. The meter’s universal power supply accepts voltages from 18 to 265V AC or DC. The meter conforms to the 1/8 DIN size standard (92 x 45 mm panel cutout) and features four 14.2 mm (0.56 inch) high red LED digits. The display is user scalable for input types other than temperature. The default L40 is set up so that a 400 Vac input reads 400. To change to another signal type or range, pry off the meter faceplate and place jumpers as illustrated in the manual. If the FS option is ordered, a specific range has been set and programmed by Laurel. Programmable features include five levels of display brightness, min and max capture, dual scaling selectable by an external control input, a deadband around 0 for AC measurements, a moving average digital filter, a count-by function, hysteresis for alarms, a fast access menu, and selectable password protection. An Option 1 board can be a single 8A relay board, an isolated active or passive 4-20 mA analog output board, or an isolated Modbus RTU RS485 serial data output board. An Option 2 single 8A relay board can plug into the Option 1 board if installed. The meter is protected to NEMA 4X (IP65) when panel mounted. It is certified to CE and RoHS standards.
