HAT-MDD10 is designed to drive two high current brushed DC motor up to 10A continuously by using Raspberry Pi as the controller. Features: -Bi-directional control for two brushed DC motor. -Support motor voltage ranges from 6VDC to 24VDC. -On board test button for manual control of motor. -Every channel output built with LED indicator. -Maximum current up to 10A continuous (without heatsink at 25℃) and 30A peak (10 second). -Current limiting at 30A. -Raspberry Pi compatible connector. -Solid state components provide faster response time. -Fully NMOS H-Bridge for better efficiency and no heat sink is required. -Speed control PWM frequency up to 20KHz (Actual output frequency is same as input frequency from Raspberry Pi). -Support sign-magnitude PWM operation. -Support TTL PWM from microcontroller, not PWM from RC receiver. -Dimension:67.5mm x 59mm -Made In Malaysia
