Compatible with Xfinity only Set Up Instructions: 1) Open Smartthings App. Click “+” to add a device. 2) Pull battery tab on device. Open device and make sure green light is flashing. I had to reinstall one battery to get the green light flashing. 3) Click scan in app. It will find it and name it “Thing”. 4) In web browser, go to Smartthings IDE 5) Go to “My devices”. Click on “Thing”. Scroll to bottom. Click “Edit”. 6) Change name and label to you name for the device. 7) For Type, select “SmartSense Open/Closed Sensor”. Many places suggest using “Tyco” but that didn’t give me the right status in the Smartthing App. 8) Scroll to bottom and click “Update”.
