This iTempo 60’ Control cable is required for the operation of iTempo controls on Mr.Steam’s Residential MS model steam bath generators with serial numbers beginning with a 900000. The iTempo 60’ Control cable connections between the generator and iTempo control. This cable is an 8-pin Mini Din which is similar to computer mouse & keyboard cables. 30’ comes as standard with a control package. The 60’ may be purchased if more than 30’ is needed. Do note though that you cannot add a 30’ plus a 60’ to make 90’, there is only one or the other, 30’ or 60'.
