Robert Sorby 3/4 Inch Deep Flute Bowl Gouge, HSS, Unhandled, Sovereign Compatible, 18" Overall Robert Sorby offers the many of their tools in unhandled versions allowing turners the flexibility to fashion their own custom wood handles This High Speed Steel (HSS) blade is the exact same as the one offered in the Sorby handled tools This HSS tool will keep a sharpened edge about six times longer than carbon steel and resist “blued” softening from overheating during grinding or from heavy use. This blade is compatible with the Sorby Sovereign System Modular handles.

Sovereign is a new handle and tool system made entirely in the Robert Sorby factory in Sheffield which allows the turner to use a wide range for gouges and hollowing tools in one comfortable handle. The handle itself comes in two lengths - 16" and 22" and there is also a coupler available for those who wish to fit two handles together. Both handles will take a tool with 5/8" shank directly. Each system comes with two collets to allow 3/8" shank tools to be fitted. There is the optional extra of a 1/2" pilot bored collet. Key features of the Sovereign System:System has been designed by an established tool manufacturer to take account of both handle and tool use to give the ultimate in comfort and balance. Handle is made from a seamless steel section. High grade aluminum head and base help to dampen vibration. Handle fitted with a comfortable non-slip Softex grip. Knurling on head an base give extra grip. Tools secured in place by brass tipped screws to avoid damage. Side handle for additional control is an optional extra. Base can be unscrewed to allow for counterbalance weight to be fitted into internal thread.
