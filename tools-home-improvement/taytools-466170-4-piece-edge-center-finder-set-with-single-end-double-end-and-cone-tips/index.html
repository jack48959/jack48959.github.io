Edge and center finders are useful for accurately finding the edge and center positions on workpieces when using milling, jig boring and engraving machines. The internal tension spring between the shaft and tip provides a flexible connection that allows the tool to work as intended. This set included 4 different edge finders: 1) Single end with 0.500 shaft and 0.200 tip, 2) Double end with 0.500 shaft, 0.200 tip and cone tip, 3) Double end with .0375 shaft, 0.200 tip and cone tip and 4) Single end with 0.375 shaft and cone tip.
